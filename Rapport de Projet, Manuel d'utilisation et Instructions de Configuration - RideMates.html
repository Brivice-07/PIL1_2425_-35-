Manuel d'utilisation de l'application web RideMates

Groupe : PIL1_2425_35
Date : Juin 2025

Présentation

RideMates est une application web développée par notre groupe pour permettre aux étudiants de l'IFRI de partager leurs trajets quotidiens entre leur domicile et le campus avec d'autres camarades.

Structure du projet

  • Gestion des comptes utilisateurs
  • Profil utilisateur détaillé
  • Publication et recherche de trajets
  • Système de mise en relation automatique
  • Messagerie instantanée entre utilisateurs

Se connecter à son compte

  1. Allez sur la page de connexion via le lien Se connecter.
  2. Entrez votre adresse e-mail ou numéro de téléphone.
  3. Entrez votre mot de passe.
  4. Cliquez sur Se connecter.
  5. Si vos identifiants sont corrects, vous serez redirigé vers la page d'accueil.
  6. Sinon, un message d’erreur apparaît.

S’inscrire pour la première fois

  1. Allez sur la page d’inscription via le lien S'inscrire.
  2. Remplissez les champs suivants :
    • Nom complet (obligatoire)
    • Adresse e-mail (doit être unique)
    • Numéro de téléphone (doit être unique)
    • Mot de passe
    • Rôle : choisissez si vous êtes conducteur ou passager
  3. Cliquez sur Créer mon compte.
  4. Une fois inscrit, vous pouvez vous connecter avec vos identifiants.

Compléter et modifier son profil

  1. Connectez-vous à votre compte.
  2. Allez sur la page Mon Profil.
  3. Vous y trouverez :
    • Votre nom et prénom
    • Votre rôle actuel
    • Vos horaires habituels
    • Votre point de départ
    • (Si conducteur) Les informations de votre véhicule et le nombre de places disponibles
  4. Cliquez sur Modifier mon profil pour éditer vos informations.
  5. Cliquez sur Enregistrer pour sauvegarder vos changements.

Publier ou chercher un trajet

Pour un conducteur :

  1. Allez sur la page Publier un trajet.
  2. Indiquez :
    • Point de départ
    • Point d’arrivée
    • Horaire de départ
    • Nombre de places disponibles
  3. Cliquez sur Publier pour que votre offre apparaisse dans la liste.

Pour un passager :

  1. Allez sur la page Rechercher un trajet.
  2. Filtrez selon :
    • Votre point de départ
    • Heure souhaitée
  3. L’algorithme vous propose des correspondances possibles.
  4. Sélectionnez un conducteur et envoyez-lui un message pour confirmer les détails.

Utiliser la messagerie instantanée

  1. Une fois connecté, cliquez sur Messages dans la navigation.
  2. Dans la liste des conversations, sélectionnez un contact pour voir l'historique des messages.
  3. Écrivez votre message dans le champ de saisie et appuyez sur Envoyer.
  4. Les nouveaux messages reçus sont visibles immédiatement.
  5. Une notification apparaît lorsqu’un nouveau message arrive (simulée ou en temps réel si implémentée).

🚀 Instructions de Configuration - RideMates

Ce guide vous aidera à configurer l'environnement de développement pour l'application RideMates.

1. Installer MySQL

Téléchargez et installez MySQL depuis : https://dev.mysql.com/downloads/

Vérifiez que MySQL est bien installé en exécutant dans votre terminal :

mysql -u root -p

2. Importer la base de données ifri_comotorage

Placez-vous dans le dossier contenant le fichier Database.sql (qui devrait être fourni avec le projet), puis exécutez la commande suivante :

mysql -u root -p < Database.sql

Cela créera automatiquement la base de données ifri_comotorage avec toutes les tables et données initiales.

3. Créer un utilisateur MySQL dédié

Connectez-vous à MySQL via votre terminal :

mysql -u root -p

Ensuite, exécutez les commandes SQL suivantes pour créer un utilisateur et lui accorder les privilèges nécessaires :


CREATE USER 'ridemate'@'localhost' IDENTIFIED BY 'ridemate123';
GRANT ALL PRIVILEGES ON ifri_comotorage.* TO 'ridemate'@'localhost';
FLUSH PRIVILEGES;

4. Modifier le fichier config.py

Dans le dossier de votre projet, naviguez vers Backend/config.py et assurez-vous que la ligne de configuration de la base de données est la suivante :


SQLALCHEMY_DATABASE_URI = 'mysql+pymysql://ridemate:ridemate123@localhost/ifri_comotorage'

⚠️ Attention : Remplacez le nom d'utilisateur (ridemate) et le mot de passe (ridemate123) si vous avez choisi d'autres identifiants lors de l'étape précédente.

5. Créer un environnement virtuel et installer les dépendances

Ouvrez votre terminal et suivez ces étapes pour installer les dépendances du backend :


cd Backend
python -m venv venv
source venv/bin/activate        # Sous Windows : venv\Scripts\activate
pip install -r requirements.txt

6. Lancer le backend Flask

Après avoir installé les dépendances, lancez le serveur Flask :


cd app
flask run

L'API sera disponible par défaut sur http://localhost:5000.

7. Lancer le frontend Vue.js

Dans un nouveau terminal, naviguez vers le dossier du frontend et lancez-le :


cd Frontend/RideMates
npm install
npm run dev

Le site sera disponible sur http://localhost:5173 (ou un autre port affiché dans la console).

🎉 Résultat

Vous pouvez maintenant tester l'application en accédant au frontend depuis votre navigateur. Assurez-vous que le backend tourne également pour que toutes les fonctionnalités soient opérationnelles.
